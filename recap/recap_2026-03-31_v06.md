## Logica quote per-parete

### Selezione pareti da quotare
1. Pareti processate in ordine **lunghezza decrescente** → le pareti principali hanno priorità
2. Skip se lunghezza pixel < **70px** (≈30-50cm a scala tipica) — evita label illeggibili su muri corti
3. Skip se la linea di quota fuoriesce dall'area bitmap (14px padding minimo)

### Direzione outward (esterno stanza)
- Centroide pixel-space calcolato come media di tutti i vertici
- Normale candidata alla parete: rotazione 90° CCW della direzione parete → `(-dz, dx)`
- Test outward: `dot(normale, midparete - centroide) > 0` → normale punta esternamente
- Se dot < 0: normale flippata → garantisce sempre proiezione verso l'esterno

### Anatomia linea di quota
```
     |←—————— 2.85m ——————→|     ← linea di quota (offset 48px outward)
     |                      |     ← tick perpendicolari (9px each side)
    / \                    / \    ← linee di raccordo (parete → quota)
===================================  ← parete
```

### Anti-crowding
- `placedLabels: MutableList<RectF>` traccia i bounding box dei label già piazzati
- Per ogni nuova label: `RectF.intersects()` contro tutti i label esistenti
- Se overlap: skip (la parete più corta viene sacrificata)
- Priorità pareti più lunghe → le quote principali sono sempre presenti

### Quote globali (invariate)
- `drawDimensionLabels()` (preesistente) mantiene le quote bounding-box (larghezza/lunghezza totale)
- Le quote per-parete sono **aggiuntive**, non sostituiscono le globali

## Rischi residui
1. **Pareti sul bordo estremo del bitmap**: la quota esce dall'area e viene skippata (comportamento corretto)
2. **Stanze con molte pareti brevi** (es. L-shape molto frammentata): alcune quote potrebbero essere skippate per anti-crowding — priorità alle pareti più lunghe
3. **`glLineWidth` / canvas**: nessun rischio, usa Android Canvas 2D (non GLES)
4. **Motore capture**: zero rischi — la funzione opera solo su `RoomExportData` post-scan

- [ ] Esporta una stanza → apri il PNG restituito
- [ ] Ogni lato principale ha la sua quota (linea + tick + label `X.XXm`)
- [ ] Le quote sono posizionate all'esterno della stanza (non sopra i muri)
- [ ] Le label non si sovrappongono tra loro
- [ ] Le pareti molto corte (< ~40cm) non hanno quote (anti-crowding e MIN_WALL_PX)
- [ ] La quota bounding-box globale (larghezza/lunghezza totale) è ancora presente
- [ ] Il testo delle quote è leggibile su sfondo bianco

### Stanza L-shape / non-convessa
- [ ] Le pareti interne del rientro hanno le quote sul lato corretto (outward verso l'esterno del rientro)
- [ ] Nessun overlap di label
